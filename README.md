# 🚀 ScanRx — Smart Medicine Safety System  

## 🧠 What is ScanRx?  
ScanRx is an AI-powered healthcare platform that helps users verify medicines, find nearby pharmacies, and get instant medical guidance using voice and AI.  
It aims to reduce fake medicines, improve accessibility, and provide quick help in emergencies.

---

## ⚠️ Problem  
- Fake medicines are a serious issue  
- Difficulty in finding nearby pharmacies with required medicines  
- Lack of quick medical guidance in urgent situations  
- Accessibility issues for visually impaired users  

---

## 💡 Solution  
ScanRx provides an all-in-one platform to:  
- Verify medicine authenticity using barcode  
- Find nearby pharmacies with availability  
- Get AI-based health assistance  
- Use voice assistant for accessibility  
- Access emergency SOS and first-aid guides  

---

## ✨ Key Features  

### 🔬 Medicine Verification  
Scan barcode to check if medicine is genuine or suspicious  

### 🏥 Pharmacy Finder  
Search medicines and locate nearby pharmacies  

### 🤖 AI Health Assistant  
Ask health-related questions and get concise answers  

### 🎤 Voice Assistant  
Speak your symptoms and get AI responses (accessible for all users)  

### 🚨 Emergency SOS  
Quick access to emergency numbers and first-aid videos  

### 👨‍⚕️ Contact Doctor  
View and connect with available doctors  

---

## 🛠 Tech Stack  

- **Frontend:** React.js  
- **Backend:** Node.js + Express.js  
- **Database:** MySQL  
- **AI:** Google Gemini API  
- **Other:** Speech Recognition API, Text-to-Speech
